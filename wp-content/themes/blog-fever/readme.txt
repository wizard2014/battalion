Blog Fever is a responsive modern design theme designed and developed by Yavor Spasov. 

The design of Blog Fever focuses on typography and presents the content front and center. Fonts are clear and large. The featured images are responsive.

== Features ==

=== Translations ===
Blog Fever ships with English as the default language.

=== Customizer Options ===
Extended options available from the Customizer:

1. Header background color
2. Header text color
3. Header image
4. Optional Header image of 1200px x 300px

=== Widgetized Areas ===
1. Right sidebar, max content width 300px.
2. Footer, max content width 320px.

=== Post Formats ===
The supported post formats are Standard and Aside.

=== Featured Images ===
Featured images are available for both posts and pages.
